Entities:

  1. User (most generic type. Maybe abstract).
  2. Admin (is a User)
  3. Employee (is a User)
  4. Direct Supervisor (DirSup) (is a User)
  5. Department Head (DeptHead) (is a DirSup)
  6. Benefits Coordinator (is a User)
  7. Tuition Reimbursement Request (contains the form)

User Stories:

USn = User Story n, where n is a positive integer.

  1. US1: a User can log in.
  2. US2: a User can log out.
  3. US3: only an Admin can create a User account.
  4. US4: only an Admin can delete any User account, except its own.
  5. US5: an Employee can fill & submit a Tuition Reimbursment Form (TRF).
  6. US6: an Employee can view their TRFs.
  7. US7: an Employee can can cancel the request if BenCo has changed the amount.
  8. US7: an Employee can view messages/notifications from BenCo, DirSup, and/or DeptHead, (maybe attached to the TRF).
  9. US8: an Employee can upload new docs to the TRF.
  10. US9: a Direct Supervisor (DirSup) can view the TRF/TRR's of the Employees assigned to them.
  11. US10: a DirSup can request additional materials/info from Employee for a TRR.
  12. US11: a DirSup can approve of deny a TRR (Tuition Reimbursement Request).
  13. US12: a Department Head (DeptHead) can view the TRF/TRR's of the Employees assigned to them.
  14. US13: a DeptHead can request additional materials/info from Employee or their DirSup for a TRR.
  15. US14: a DirSup can view the Employee's Presentation for Confirmation of Passing (CoP)
  16. US15: a Benifits Coordinator (BenCo) MUST approve/deny a TRR.
  17. US16: a BenCo can view the Employee's Grade for CoP.
  18. US17: a BenCo can request additional materials/info from Employee, their DirSup, or their DeptHead.
  19. US18: a BenCo can change the reimbursment amount.
  20. US19: a BenCo can view their assigned Employee's TRRs.
